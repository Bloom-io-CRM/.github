## Everyday Fixes and Adjustments

Client cannot find a document: confirm the project is linked to the correct Bloom.io client portal record and resend the portal access email. Payment not matching a job: review Bloom.io payments and invoice status inside the client project before creating a duplicate invoice.

Booking link not showing the right times: check Bloom.io scheduling availability, service duration, buffer settings, and connected calendar rules. Contract not ready to send: update Bloom.io contracts templates, confirm required fields, and test Bloom.io contract signing with an internal contact before sending it to a client.

Lead details feel incomplete: refine Bloom.io lead management forms and add required intake questions. If a service page needs stronger presentation, update Bloom.io portfolio website content so prospects see packages, examples, and booking options before they contact you.
